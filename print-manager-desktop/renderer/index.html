Configurar Impressora Virtual no Windows

Passo a Passo:

  1. Abra Configurações → Dispositivos → Impressoras e scanners
  2. Clique em Adicionar impressora ou scanner
  3. Selecione Microsoft Print to PDF
  4. Clique com botão direito na impressora → Propriedades da impressora
  5. Vá em Avançado → Padrões de impressão
  6. Configure para salvar PDFs na pasta: C:\CacimboPrint\Queue
  7. No seu ERP, selecione esta impressora para imprimir
